Welcome to PawNode
PawNode is an experimental, currently IPv6-only, Anycast network
Currently we operate PoPs in Seattle, New Jersey, Vancouver and Frankfurt
For any inquiries, please contact hostmaster@pawnode.com

ASN

  • AS207618 (Securebit)

IPv4

  • None

IPV6

  • 2a09:4c2:1a::/48 (Securebit)
  • 2a0e:7d44:f000::/40 (Red Panda Network)
  • 2a0f:9400:7310::/44 (Free Range Cloud)

Peering requirements:

  • Public ASN
  • Available IPv6 space (at least /48) to use
  • Only route traffic destined to the PawNode network

Connectivity:

  • EVIX
  • nmetered
  • GRE or other tunnel
For peering inquiries contact peering@pawnode.com
